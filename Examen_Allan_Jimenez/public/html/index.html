Triwago

Que es Triwago?

Triwago es una aplicación para el registro, visualización y evaluación de hoteles, para facilitarle al usuario la búsqueda para de un lugar para hospedarse en sus viajes.

Objetivos

Facilitarle al usuario la evaluación de hoteles para tener un control con un ranking todos lo hoteles registrados.

Permitir facil accesso para el registro de usuarios y evaluacion de hoteles.
